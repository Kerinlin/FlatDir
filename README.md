# electron-vue
![GIF.gif](https://i.loli.net/2020/12/25/qKiAN9JvUWV2moE.gif)

将目录下的所有文件(包括子文件夹内的文件)全都提取到一个总目录中,对于重名文件会检测大小，更改文件名
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn electron:serve
```

### Compiles and minifies for production
```
yarn electron:build
```
